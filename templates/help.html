Tutorial

How to organize the folder structure?

The dataset to be uploaded to Google drive must have the following structure:

/root |- /input | |- /train | | | - $image1$.{tiff, png, jpg, etc.} <--------+ | | | - $image2$.{tiff, png, jpg, etc.} | | | | - ... | | | | - $imageN$.{tiff, png, jpg, etc.} | | | | | |- /test | | | - $image1$.{tiff, png, jpg, etc.} |=====> Same file name (no `WHITE`, `F1`, `F2`, etc.) | | - $image2$.{tiff, png, jpg, etc.} | | | - ... | | | - $imageN$.{tiff, png, jpg, etc.} | | | |- /output | |- /train | | |- $image1$.{tiff, png, jpg, etc.} <--------+ | |- ... | |- $imageN$.{tiff, png, jpg, etc.} | |- /test | - $image1$.{tiff, png, jpg, etc.} | - $image2$.{tiff, png, jpg, etc.} | - ... | - $imageN$.{tiff, png, jpg, etc.}

Example dataset can be download from Google drive.
There is a list of available datasets in cell-db. However, that database is using slightly different naming convention. Basically, you need to change folder "A" to "input", and folder "B" to "output".

What is the size (in pixels) of the images?

Currently we only accept 3-channel RGB images of size 1024x1024 pixels. In the future, we may be more flexible.

What kind of image formats do you accept?

We accept a wide variety of formats, including *.tiff, *.png, *.jpg, etc. A full list can be found here. However, note that the prediction results will always be in *.png format.

Where to find my google-drive link?

First, compress your root folder to a zip file (e.g. data.zip); then upload it to your personal google-drive. Make sure you opened the file sharing permission! This can be done by right-click the file -> "Get shareable link". See images below:

How long does it take to get my prediction results?

It depends on the size of your dataset, for example it takes 4-6 hours if the training set contains 300-500 images.

How to choose the training hyper-parameters?

In most cases, the default settings are good enough. This default settings have been tested by tens of experiments I conducted before. Most likely, it will also work in your dataset. One exception is the Ensemble size, you are always free to select more than one models.

Where can I get the download link?

There are two places to download the results. First, once the training is completed, you will receive an email like this

Your model and predictions are ready
Your model and predictions can be downloaded from https://f002.backblazeb2.com/file/MSC-project/fnet-web/xxxxx.tar.gz
The other way is to visit Jobs tag, find the correct experiment ID and name, if the state is SUCCESS, you will see a download button.
